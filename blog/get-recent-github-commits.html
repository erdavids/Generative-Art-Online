How to Display Recent Github Commits with Javascript

2 min read - 7/3/19


When people visit my site, I want them to be aware that I'm working on new projects and have an easy way of following my recent progress. Obviously they would see all of that if they visit my Github profile, but I didn't want to waste the opportunity of my first impression with this visitor.

Once I started rolling with the idea of displaying commits on my home page, I was pleasantly surpised by how simple it was to set up.

RSS Feed

Luckily for me, Github helps out by providing an RSS feed for both user accounts and individual repositories. It took me a little while to hunt down definitive info on these, all you have to do is add .atom to feed you would like to access.

> Feed for my Github profile - https://github.com/erdavids.atom > Feed for single repository - https://github.com/erdavids/Oblong/commits/master.atom

JQuery

You'll be using Ajax to grab any recent activity, so you'll need to make sure you are using JQuery.

The easiest way is to add a snippet from Google's hosted libraries listed under JQuery.

Enhancements

My advice would be to copy all this code into your own environment and start playing around with the request response. There's a lot of cool info on your Github activity in those feeds that you can use to add a lot of personality to your portfolio.
