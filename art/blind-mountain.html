Blind Mountain

This project ended up being a lot cooler than I expected. I actually didn't plan using any colors other black and white, but once I started playing around, the project immediately took a new form. Setting the two tone colors was also a complete mistake, but I decided to keep it.

While each line is being drawn, it has a random chance of entering a deformation state. Once that happens, a different random event is required for it to go back to normal. While deformed, the line is simply curving through deviated points that become more and more exagerrated. If you want to learn more about how it's created, check out this repository.
