Space Invaders

Generate random symmetrical sprites of any size.

I started this project to learn more about generative art. I was intrigued by the idea of using code to create something tangible and artistic.

Overall the logic behind it is very simple. Each sprite has odd numbered dimensions and consists of randomly chosen colors placed symmetrically. Each image is statistically one of a kind, and messing around with some of the options in the code can create some extremely interesting results.
