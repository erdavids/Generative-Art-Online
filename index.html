Space Invaders

Generate random symmetrical sprites of any size.

I started this project to learn more about generative art. I was intrigued by the idea of using code to create something tangible and artistic.

Overall the logic behind it is very simple. Each sprite has odd numbered dimenposions and consists of randomly chosen colors placed symmetrically. Each image is statistically one of a kind, and messing around with some of the options in the code can create some extremely interesting results.

Queue by Queue

An increasingly difficult video game with unique memory and puzzling mechanics.

At first glance, this game is extremely simple. Move the large blue squares to the small ones by utilizing the effects of the different colored squares. There is another aspect of difficulty that makes it fairly unique.

In addition to understanding all the different mechanics of the tiles, the player must solve the entire puzzle without seeing anything move. All of action inputs are placed in a queue and nothing happens on the board until the player 'executes' the queue. It can be surprisingly difficult to keep up with the state of the board, especially when certain tiles change the layout.
